# Docker Hub

Docker Hub is the official online repository for Docker images. It allows users to download, upload, and manage container images.

### Features

* Official and community images
* Public and private repositories
* Image versioning
* Easy image sharing

**Website**

```text id="k92mxd"
https://hub.docker.com
```

# Docker Images

A Docker Image is a template used to create containers. It includes everything required to run an application.

**Popular Images**

* Ubuntu
* Nginx
* Apache
* MySQL
* Redis
* Python
* Node.js
* MongoDB

# Working with Images

Search for an image:

```bash id="m4t1ya"
docker search nginx
```

Download an image:

```bash id="z5l0xq"
docker pull nginx
```

View downloaded images:

```bash id="w7cn9p"

```

# Port Binding

Port Binding connects a container port with a host machine port so that the application can be accessed from a browser.

**Syntax**

```bash id="v9kj2n"
docker run -p <host-port>:<container-port> <image-name>
```

**Example**

```bash id="g8ta6s"
docker run -p 8080:80 nginx
```

**Explanation**

* **8080** → Host machine port
* **80** → Container port

---

# Running Nginx with Port Binding

```bash id="b4rm8j"
docker run -d -p 8080:80 --name my-nginx nginx
```

**Options Used**

* `-d` → Run in background
* `-p` → Port mapping
* `--name` → Assign container name
* `nginx` → Docker image

---

# Accessing the Application

Open the browser and enter:

```text id="e6yh1f"
http://localhost:8080
```

For an AWS EC2 instance:

```text id="j3pw7v"
http://<EC2-Public-IP>:8080
```

> **Note:** Ensure that port **8080** is allowed in the EC2 Security Group.

# Useful Docker Commands

| Command                   | Purpose                 |
| ------------------------- | ----------------------- |
| `docker ps`               | List running containers |
| `docker ps -a`            | List all containers     |
| `docker stop my-nginx`    | Stop a container        |
| `docker start my-nginx`   | Start a container       |
| `docker restart my-nginx` | Restart a container     |
| `docker rm my-nginx`      | Remove a container      |
| `docker rmi nginx`        | Remove an image         |
