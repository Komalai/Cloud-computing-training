# Introduction to Docker

Docker is an open-source platform used to develop, package, and run applications inside containers. It helps applications run consistently across different systems.

### Advantages

* Lightweight
* Fast deployment
* Platform independent
* Efficient resource usage
* Easy application management

---

# What is a Container?

A container is a lightweight environment that includes everything an application needs to run.

It contains:

* Application code
* Runtime
* Libraries
* Dependencies
* Configuration files

---

# Virtual Machine vs Docker Container

| Virtual Machine     | Docker Container      |
| ------------------- | --------------------- |
| Runs on Hypervisor  | Runs on Docker Engine |
| Includes Guest OS   | Shares Host OS Kernel |
| Heavyweight         | Lightweight           |
| Slower startup      | Faster startup        |
| Uses more resources | Uses fewer resources  |

# Installing Docker (Ubuntu)

**Steps**

1. Update packages.

```bash
sudo apt update
```

2. Install Docker.

```bash
sudo apt install docker.io -y
```

3. Start Docker service.

```bash
sudo systemctl start docker
```

4. Enable Docker at boot.

```bash
sudo systemctl enable docker
```

5. Check Docker version.

```bash
docker --version
```

6. Verify installation.

```bash
docker run hello-world
```

# Basic Docker Commands

| Command               | Purpose                    |
| --------------------- | -------------------------- |
| `docker --version`    | Check Docker version       |
| `docker info`         | Display Docker information |
| `docker pull ubuntu`  | Download an image          |
| `docker images`       | List downloaded images     |
| `docker search nginx` | Search images              |

---

# Docker Container Commands

| Command                                | Purpose                     |
| -------------------------------------- | --------------------------- |
| `docker ps`                            | List running containers     |
| `docker ps -a`                         | List all containers         |
| `docker run ubuntu`                    | Run a container             |
| `docker run -d nginx`                  | Run container in background |
| `docker run --name mycontainer ubuntu` | Create a named container    |
| `docker start mycontainer`             | Start a container           |
| `docker stop mycontainer`              | Stop a container            |
| `docker restart mycontainer`           | Restart a container         |
| `docker inspect mycontainer`           | View container details      |
| `docker logs mycontainer`              | View container logs         |
| `docker exec -it mycontainer bash`     | Access container terminal   |
| `docker rm mycontainer`                | Remove a container          |
| `docker rmi ubuntu`                    | Remove an image             |
