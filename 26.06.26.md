# Interview Questions

### 1. What is Cloud Computing?

Cloud computing is the delivery of computing resources such as servers, storage, and databases over the internet on a pay-as-you-go basis.

### 2. What is the difference between IaaS, PaaS, and SaaS?

* **IaaS:** Provides virtual infrastructure like servers and storage.
* **PaaS:** Provides a platform to build and deploy applications.
* **SaaS:** Provides ready-to-use software over the internet.

### 3. What is IAM?

IAM (Identity and Access Management) is an AWS service used to manage users, groups, roles, and permissions securely.

### 4. What is an EC2 Instance?

An EC2 instance is a virtual server in AWS used to run applications.

### 5. What is an S3 Bucket?

An S3 bucket is a container used to store files (objects) in Amazon S3.

### 6. What is Amazon VPC?

Amazon VPC is a private virtual network that securely hosts AWS resources and controls network communication.

### 7. Difference between Security Group and Network ACL?

| Security Group            | Network ACL                        |
| ------------------------- | ---------------------------------- |
| Works at instance level   | Works at subnet level              |
| Stateful                  | Stateless                          |
| Supports only Allow rules | Supports both Allow and Deny rules |
