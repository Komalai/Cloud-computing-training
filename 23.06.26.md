# Amazon EC2

Amazon EC2 (Elastic Compute Cloud) is an AWS service used to create and manage virtual servers (instances) in the cloud.

### Features

* On-demand virtual machines
* Scalable resources
* Pay-as-you-go pricing
* Supports multiple operating systems
* CloudWatch monitoring

---

# EC2 Components

### Instance

A virtual machine running in AWS.

### AMI (Amazon Machine Image)

A pre-configured image used to launch an EC2 instance.

**Examples**

* Amazon Linux
* Ubuntu
* Windows Server
* Red Hat

### Instance Type

Defines CPU, memory, and storage capacity.

**Examples**

* t2.micro
* t3.micro
* t3.small

### Key Pair

Used for secure login to an EC2 instance.

* Public key stored by AWS
* Private key downloaded by the user

### Security Group

Acts as a virtual firewall to control network traffic.

# Launching an EC2 Instance

Steps performed:

1. Open AWS Management Console.
2. Go to **EC2 → Launch Instance**.
3. Enter instance name.
4. Select AMI.
5. Choose instance type.
6. Create a Key Pair.
7. Configure network settings.
8. Add storage.
9. Click **Launch Instance**.

**Example Configuration**

```text
Name: MyFirstServer
AMI: Amazon Linux 2023
Instance Type: t2.micro
Storage: 8 GB SSD
```

# Security Group Rules

| Type  | Port | Purpose        |
| ----- | ---- | -------------- |
| SSH   | 22   | Remote Login   |
| HTTP  | 80   | Website Access |
| HTTPS | 443  | Secure Website |

---

# EC2 Instance States

| State      | Description                  |
| ---------- | ---------------------------- |
| Pending    | Instance is starting         |
| Running    | Instance is active           |
| Stopped    | Instance is powered off      |
| Stopping   | Shutdown in progress         |
| Terminated | Instance deleted permanently |

# Connecting to EC2

### Browser

* Select the running instance.
* Click **Connect**.
* Choose **EC2 Instance Connect**.

### SSH

```bash
chmod 400 mykey.pem
ssh -i mykey.pem ec2-user@public-ip
```

# Hosting a Web Page

Update the system:

```bash
sudo yum update -y
```

Install Apache:

```bash
sudo yum install httpd -y
```

Start Apache:

```bash
sudo systemctl start httpd
sudo systemctl enable httpd
```

Create the homepage:

```bash
echo "<h1>Hello from AWS EC2</h1>" | sudo tee /var/www/html/index.html
```

---

# Accessing the Website

* Allow **HTTP (Port 80)** in the Security Group.
* Copy the instance's **Public IPv4 Address**.
* Open in browser:

```text
http://Public-IP
```
